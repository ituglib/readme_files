Instructions for use of Xz versions 5.4.4 and above

Use the OSS command
  tar xzf xz-package.tar.gz -C / -p --no-same-owner
to install xz.

Due to new features added in the latest OSS System Libraries, binaries built with those will not function correctly on older RVUs. The following table lists the minimum RVU for each platform that can correctly use xz version 5.4.4, and above:

Platform Minimum RVU
TNS/E J06.21
TNS/X L17.08
TNS/V L17.08

If you require new features or bug fixes that were released after 5.4.4, you will have to download the source from the xz download and build the new version yourself. This requires experience with FLOSS and with building open source tools on OSS. Make sure to replace the line in src/xz/file_io.c that includes poll.h with the following:

#include <poll.h>
#define _FLOSS_USE_POLL
#include <floss.h(floss_poll)>
