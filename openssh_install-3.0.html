Note: The port of OpenSSH 8.xp1 may not be fully functional. Key generation and signing appear to work correctly. ITUGLIB will not support the use of OpenSSH for communication. We highly recommend using NonStop SSH for communication use cases.

Instructions for use of OpenSSH 8.xp1

Use the OSS command
  tar xzf openssh-ssl3.0-version-systype.tar.gz -C / -p --no-same-user
to install OpenSSH. You must install the package in the exact same location as in the tar file. Add the /usr/local/lib to your _RLD_LIB_PATH variable when use OpenSSL 3.0 for this package. Other versions of OpenSSH should be installed elsewhere.

Due to new features added in the latest OSS System Libraries, binaries built with those will not function correctly on older RVUs. The following table lists the minimum RVU for each platform that can correctly run OpenSSH version 8.9p1, and above:

Platform Minimum RVU
TNS/E J06.22
TNS/X L20.10
TNS/V L20.10

The source for this project is currently in a fork available at: https://github.com/ rsbeckerca/openssh-portable.git. This location is subject to change if and when the contribution to the main project is accepted.
