Instructions for use of ncurses versions 6.5 and above

Use the OSS command
  tar xzf ncursespackage.tar.gz -C / -p --no-same-owner
to install ncurses. If you are using the ncurses DLLs, make sure to have the directory containing the appropriate DLL in your _RLD_LIB_PATH variable when ncurses is used. Typically, this is /usr/local/lib or /usr/local/lib64.

If you are building 64-bit ncurses yourself, you will need the c11 compiler and a minimum RVU of L20.10. Fix any compile errors that occur. You may have issues in configure with intptr_t and in search.h.

Due to new features added in the latest OSS System Libraries, binaries built with those will may not function correctly on older RVUs. The following table lists the minimum RVU for each platform that can correctly run ncurses version 6.5, and above:

Platform Minimum RVU
TNS/E not available from ITUGLIB
TNS/X L20.10
TNS/V L20.10
