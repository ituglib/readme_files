Instructions for use of jgit versions 7.3.0 and above

Use the OSS command
  tar xzf jgit-package.tar.gz -C /usr/local -p --no-same-owner
to install jgit. Make sure to have the directory containing the appropriate Java packages (Java 17+) set up properly to use jgit.

Due to new features added in the latest Java 17, older Java versions are not supported. The following table lists the minimum RVU for each platform that can correctly run jgit version 7.3.0, and above:

Platform Minimum Java Notes
TNS/X 17
TNS/V 17
TNS/E n/a JGit cannot be run on TNS/E because Java 17 is required.
